CSS Selectors

We have some CSS selectors to style the html elements.

  • 1)Universal selectors (*)
  • 2)Element selectors (tag name)
  • 3)class name selectors (class name using)
  • 4)Id selectors (using id attribute #)
  • 5)attribute selectors (using any attributes inside the elements)
  • 6)attribute with value selector (attribute with the value)
