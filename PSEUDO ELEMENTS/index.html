CSS Pseudo-Elememts

The Css Pseudo-Elememts allow you to style the elements or parts of the elements without adding any IDs or classes to them.

CSS3 introduced a new double-colon (::) syntax for Pseudo-Elememts to distinguish between them and Pseudo-Classes.

Selector::Pseudo-Elememt { property: value; }
